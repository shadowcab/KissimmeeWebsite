jQuery Methods for Forms and Controls (Ruvalcaba and Delamater)

val()

a jQuery method that is used with a $(“selector”). Without specifying an argument, val() will retrieve the value of a form element, such as a text box, and set it to the associated declared variable. If you specify an argument, such as val(“Hello World”), the text will be set for the form element referenced by the selector.

text()

is similar to val() except it sets and gets the inner content of regular elements (not form elements). It is also similar to html(), but text() performs a literal translation of the text including HTML tags.

next()

is a jQuery traversing method used to reference the next sibling of the selector reference $(“selector”). It can also reference the first sibling of a particular element if an argument is specified and that element type immediately follows the selector reference, ex. $(“div”).next(“p”). This example will reference the first p element that immediately follows a div block.

submit()

is used to initiate the processing of a form, as referenced by the selector, by submitting the data to the server.

focus()

is used to set the focus to a form control or link referenced by the selector.


General Purpose jQuery Methods (Ruvalcaba and Delamater)

prev()

a jQuery traversing method similar to next(), but it provides a reference to the previous sibling element to the selector reference. If an argument element type is specified, then the previous element of the argument type is referenced as long as it precedes the selector reference.

find(filter)

is a jQuery traversing method, used to reference a descendant element of the selector element. The descendant element is specified by the required argument of the find() method (“JQuery Traversing Methods”).

attr()

is a jQuery method for setting or getting HTML and CSS attribute values of the selected element. There are several overloaded versions: get or set a single attribute value or multiple attributes and values.

css()

is a jQuery method for setting or getting CSS style properties of the selector element. The applied CSS is specified by the arguments of the css() method. The first argument specifies the CSS property name and the second argument specifies the CSS property value.

addClass()

is a jQuery method for adding one or more class names to a selected element. You must specify one parameter for this method: the class selector name. As a best practice, it's better to use class selectors to apply styles as opposed to utilizing the css() method multiple times.

removeClass()

is a jQuery method for removing one or more class names from the selected element.

toggleClass()

provides a means to toggle on and off one or more class names for the selected element.

html()

is used to set or get the innerHTML of the selected element. Unlike text(), this method will translate HTML tags.

hide()

is a jQuery “effects” method. It is used to hide (that is, make disappear) the selected element. Various arguments allow you to specify the speed and transition for the vanishing element.

show()

is the opposite of hide(). It is used to reveal the selected element that was previously hidden.

each(function)

is a jQuery traversing method, used to apply a function to each element in the array of selected elements. The function is required (“JQuery Traversing Methods”).


jQuery Event Methods (Ruvalcaba and Delamater)

ready()

is a jQuery event method, used to load any JavaScript after the DOM has been loaded. This safeguards any JavaScript from executing before the DOM is created.

click()

triggers the click event or attaches a JavaScript function to the click event on the selected element.

mouseover()

attaches a trigger and callback function to the mouseover event for the selected element. The function is also applied to any of the selected element’s children (“JQuery Event Methods”).

hover()

attaches a trigger and callback function to the mouseenter and mouseleave events for the selected element. If only one function is specified, then the same function is executed for both events (“JQuery Event Methods”).


jQuery Effect Methods (Ruvalcaba and Delamater)

animate({styles},speed,easing,callback)

morphs or animates the numeric CSS properties of an element.

delay(duration in ms)

delays or pauses the execution of subsequent jQuery effects or animations following the dot notation order. Example: $("#idSelector").slideUp(1000).delay(500).fadeIn(200);

stop()

immediately terminates a jQuery animation or effect.

finish()

immediately terminates a jQuery animation or effect and sets the CSS properties to their target vales. All queued animations are also terminated.
